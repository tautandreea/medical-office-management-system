# Medical Office Management System

Full-stack application for managing the activities of a medical office.

## Description

This project was developed as an academic software engineering project.  
The application allows users to manage doctors, patients, appointments, medical records, statistics and notifications.

## Technologies Used

- Java
- Spring Boot
- React
- MySQL
- Maven
- REST APIs
- Microservices architecture

## Main Features

- User authentication
- Patient management
- Doctor management
- Appointment scheduling
- Medical record management
- Statistics and export functionality
- User notifications
- API Gateway for communication between services
- Frontend interface built with React
